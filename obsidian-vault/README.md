# 📒 Workspace Vault

Welcome! This is your Obsidian **vault** — a folder of plain Markdown (`.md`) files
that Obsidian turns into a connected, searchable knowledge base. It's pre-configured
to help you run your two active projects: **One Portal** and **NHB Consultancy**.

---

## 🚀 First time using Obsidian? Start here (5 minutes)

1. **Open the vault**
   - Open Obsidian → *Open folder as vault* → select this `obsidian-vault` folder.
   - When asked to "Trust author and enable plugins", choose **Trust** (this vault only
     uses Obsidian's built-in core plugins — no third-party code).

2. **Go to your home base**
   - Open `00 Dashboard/Home.md`. This is your hub — every project, template, and
     section links from here. Click any `[[link]]` to jump around.

3. **Learn the three things that matter most**
   - **Links:** type `[[` and start typing a note name to link notes together.
   - **Search:** `Ctrl/Cmd + O` jumps to any note instantly. `Ctrl/Cmd + Shift + F`
     searches all text.
   - **Templates:** `Ctrl/Cmd + P` → "Insert template" to drop in a ready-made
     meeting note, project page, etc.

That's genuinely enough to be productive. Everything below is reference.

## 🗂️ How this vault is organised

A lightweight version of the **PARA** method (Projects / Areas / Resources / Archive):

| Folder | What goes here |
|--------|----------------|
| `00 Dashboard` | Your home page + project index. Start every session here. |
| `01 Projects` | Active, outcome-driven work with a deadline. **One Portal** and **NHB Consultancy** live here. |
| `02 Areas` | Ongoing responsibilities with no end date (e.g. Finance, Admin, Personal). |
| `03 Resources` | Reference material, research, useful links, reusable docs. |
| `04 Archive` | Finished or paused projects. Move things here instead of deleting. |
| `05 Daily Notes` | One note per day for quick logging. Created automatically. |
| `Templates` | Reusable note skeletons. Don't edit these for actual content — insert them. |

> **Rule of thumb:** if it has a finish line, it's a Project. If it's "just part of
> the job, forever", it's an Area.

## ✅ A simple daily workflow

1. Open today's **Daily Note** (`Ctrl/Cmd + P` → "Open today's daily note").
2. Jot tasks, calls, and ideas there as they happen — don't worry about filing.
3. When something belongs to a project, link it: `[[One Portal]]` or `[[NHB Consultancy]]`.
4. Once a week, run the **Weekly Review** template to tidy up and plan ahead.

---

## ⚙️ What's already set up for you

- **Templates** plugin → folder set to `Templates`.
- **Daily Notes** plugin → saves to `05 Daily Notes`, uses the Daily Note template.
- **Backlinks, Outline, Tag pane, Quick switcher, Search** → all enabled.
- Attachments (pasted images, etc.) are kept in a `_attachments` folder so they don't
  clutter your notes.

You can change any of this in **Settings (⚙️)** at any time.

---

## 💡 Tips

- **Tags** (`#urgent`, `#waiting`, `#idea`) cut across folders — handy for status.
- **`- [ ] task`** creates a checkbox. Search `task-todo:` or use the Tasks workflow
  in each project's `Tasks.md`.
- Don't over-organise early. Capture first, file later. The search is fast enough that
  perfect folders don't matter much.

Happy note-taking! 🎉
